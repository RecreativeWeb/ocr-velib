# Velib
Le projet 3 de ma formation DWJ sur OpenClassRooms

## Le projet
Concevoir une carte interactive de location de vélos de la ville de Paris

### Utilisation
Cloner le projet

Installer les dépendances avec [yarn](https://yarnpkg.com/en/docs/install)

```javascript
yarn install
```
puis 

```javascript
yarn run build
```
